# A simple shell

这是一个由c语言实现的简易shell

### screenshot
![demo](./img/demo.png)

### 主题配置
```shell
THEME_PREFIX_CHAR λ
THEME (white;none){hostname} (dark_green;none){directory} (purple;none){git_info}
THEME (green;none){prefix}
```
